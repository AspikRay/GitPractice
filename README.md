# GitPractice

Git Practice Project

    Репозиторий создан для отработки навыков работы с системой контроля версий Git. В рамках проекта реализован полный цикл разработки: от инициализации репозитория до разрешения конфликтов слияния и работы с Pull Requests.

Вариант 2. Структурированный (удобно для преподавателя)

    Обучающий проект: Основы Git

    Цель: Практическое освоение инструментов командной разработки.

    Реализованный функционал:

        Создание локального и удаленного репозиториев.

        Управление ветками (main, feature-1).

        Работа с историей коммитов.

        Создание и мерж Pull Requests на GitHub.

        Ручное разрешение конфликтов слияния.
